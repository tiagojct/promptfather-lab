# Prompts de exemplo — The Promptfather

Prompts para usar com o **opencode** nos exercícios. Adapta ao teu caso.

## Explorar um dataset (crítico)

```
Lê `coorte-clinica.csv`. Diz-me quantas linhas e colunas tem, os nomes das
colunas e o tipo de cada uma (numérica, categórica, data). Não avances até
teres a certeza dos tipos.
```

```
No `coorte-clinica.csv`, identifica problemas de qualidade dos dados:
valores em falta (por coluna e concentração), categorias inconsistentes,
valores implausíveis (idades, fev1), e formatos de data mistos. Lista cada
problema com exemplos concretos das linhas afetadas.
```

## Literatura

```
Procura na literatura cinco artigos recentes (últimos cinco anos) sobre o
impacto da poluição do ar urbana na função pulmonar em crianças. Para cada
um: Autores, Ano, Título, Revista, DOI. Marca claramente o que não
conseguiste verificar.
```

## Conversão e escrita

```
Converte `coorte-clinica.csv` para Excel (.xlsx) mantendo os tipos.
Depois, a partir da análise que fizemos, escreve um parágrafo de Métodos
(estilo científico, ~120 palavras) a descrever a amostra e a limpeza de dados.
```

## Regras de ouro (a IA pode errar)

- Pede tipos e contagens antes de conclusões.
- Desconfia de DOIs e referências: confirma sempre.
- Nunca envies dados identificáveis para a cloud. Para isso, usa um modelo local.
